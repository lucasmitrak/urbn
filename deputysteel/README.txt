Deputy Steel is named after the Mister Gutsy robots from Fallout.

Deputy Steel is the facebook scraper.
It grabs the data from facebook catfish and the 2nd degree data.
It puts it into a local database.
It then sends the data via ssh to the main server.

This is to be run on a node.
This should be run on a node where the catfish is managed.
You may write the data through a ssh tunnel if desired.
